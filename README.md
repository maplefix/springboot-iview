# 关于springboot-iview

## 项目介绍

  springboot-iview后端基于spring boot2.1.4版本+iview3.4开发。
  用于快速开发的一套脚手架项目。
  预期功能： 
  1. 字典维护 
  2. 菜单管理 
  3. 角色管理 
  4. 用户组织 
  5. 消息管理 
  6. 日志记录 
  7. 代码生成
  
## 目录结构

<pre>
|-iview-core 后端springboot工程
|-iview-web iview前端工程
|-iview-doc 文档目录
|-iview-generator 代码自动生成工程
</pre>
